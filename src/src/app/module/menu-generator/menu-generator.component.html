About The Project

Open Delivery - Menu Generator is a tool that provides a graphical view of a MERCHANT following the Merchant Schema of the Open Delivery standard.

Usage:

Copy your Merchant JSON schema following the Open Delivery standard specification into the Textarea and click Generate Menu.
If your JSON is not following the schema, you may get errors.

Alternatively, you can choose some ready-made JSON examples to check their structure and graphical view.
